heatmap

MakieCore.heatmap — Function
heatmap(x, y, values)
heatmap(values)

Plots a heatmap as an image on x, y (defaults to interpretation as dimensions).

source

Examples

using CairoMakie


f = Figure()
Axis(f[1, 1])

xs = LinRange(0, 10, 25)
ys = LinRange(0, 15, 25)
zs = [cos(x) * sin(y) for x in xs, y in ys]

heatmap!(xs, ys, zs)
